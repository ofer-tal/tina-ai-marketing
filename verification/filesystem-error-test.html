Error Types Classified:

  • PERMISSION_DENIED - User lacks required permissions
  • FILE_NOT_FOUND - Requested file doesn't exist
  • DISK_FULL - Insufficient disk space
  • INVALID_PATH - Malformed file path
  • FILE_LOCKED - File in use by another process
  • DIRECTORY_NOT_FOUND - Target directory missing
  • IO_ERROR - General I/O failure

Recovery Strategies:

  • RETRY - Exponential backoff (max 3 attempts, 1s-10s delays)
  • CREATE_DIRECTORY - Auto-create missing directories
  • USE_ALTERNATIVE_PATH - Fallback to alternative location
  • CLEANUP_AND_RETRY - Clean up and retry operation
  • NOTIFY_USER - Alert user of unrecoverable error
  • SKIP - Skip operation due to error
